# DeNovo-Genome-Assembly-Mycoplasmoides
De novo genome assembly and annotation of Mycoplasmoides genitalium using Illumina paired-end reads. Includes full workflow: FastQC, Fastp trimming, Seqkit stats, SPAdes &amp; Velvet assemblies (k=29, k=31), Prokka annotation, and BLAST validation. Scripts only — no raw FASTQ data included.
